# Step 5 — Evaluate and Visualize Detection Results

This folder contains a clean subset of evaluation outputs generated by `code/code_4.py`.

## What is included
- `evaluation_metrics.csv`: precision/recall/F1/FPR and dataset size.
- `confusion_matrix.csv`: row-level weak-label confusion matrix values.
- `figures/*_overlay.svg`: original time-series overlaid with red anomaly flags.
- `figures/confusion_matrix_weak_labels.svg`: visual confusion matrix.
- `predictions.csv`: per-row anomaly predictions and scores.

## Interpretation guide
- In each overlay figure, the blue line is the original signal and red points are flagged anomalies.
- Weak labels come from folder origin (`normal` vs `anomalies`) and should be interpreted as approximate ground truth.
